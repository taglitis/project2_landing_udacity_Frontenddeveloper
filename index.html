DATA SPINNING

    Data analyst

    Data engineer is a professional who is involved in the process of data preparation, ETL (data extraction, transformation and load). Data engineers usually work side by side with Data Scientists. A new direction of data engineers gain its popularity and it is called a Data Cloud Engineer. They work in cloud such a AWD, Google Cloud and in other clouds available in the world.

    Data Engineer

    Data Engineer is a person who creates, transforms, extract, load data. It is compliacted and exciting area where one need to know different technology from DATA BASES to HADOOP, from INFORMATICA to AWS. If you like to work with data and perform different transformation with it than you can consider to become Data Engineer.

    Data scientist

    Data science is a new area. It is very interesting and the main goal is by using machine learning and deep learning technics to make some prediction. For example, it is a common practice to make prediction of the next best offer in e-commerce companies and offer a product to the client wich will be bought with the high probability.
